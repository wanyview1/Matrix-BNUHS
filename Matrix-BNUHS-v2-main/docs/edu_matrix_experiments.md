# EDU-MATRIX 实验设计

## 1. 对比实验设计

### 基线方案

| 基线 | 说明 | 来源 |
|------|------|------|
| **Agent-Centric** | 传统规则硬编码到智能体 | 现有开源方案 |
| **Static Rules** | 无MLEP，静态知识库 | 对比MLEP |
| **External Filter** | 外部过滤器做对齐 | 对比内生对齐 |
| **Random** | 随机行为 | 下限基线 |

### 评估指标

| 指标 | 描述 | 目标 |
|------|------|------|
| Dialogue Consistency | 对话一致性 | >90% |
| Social Clustering | 社会聚类系数 | >0.7 |
| Value Alignment | 价值对齐率 | >85% |
| Response Quality | 响应质量 | >4.0/5 |
